-----
### 정리
-----
1. 자주 사용하는 연산자
   - 산술 연산자: +, -, *, /, % (나머지)
   - 증가 및 감소 연산자: ++, --
   - 비교 연산자: ==  !=, >, <, >=, <=
   - 논리 연산자: && (AND), || (OR), ! (NOT)
   - 대입 연산자: =, +=, -=, *=, /=, %=

2. 다음 연산자들도 자주 사용
   - 삼항 연산자: ? :
   - instanceof 연산자: 객체 타입을 확인
   - 그 외: new, [] (배열 인덱스), . (객체 멤버 접근), () (메소드 호출)

3. 비트 연산자는 실무에서 거의 사용할 일이 없음 필요할 때 찾아볼 것
   - 비트 연산자: &, |, ^, ~, <<, >>, >>>
